# Smuler Registry

Community plugin and theme index for [smuler](https://github.com/lum1n/smuler).

## Browse

```bash
smuler plugin registry list
smuler plugin registry search github
smuler theme registry list
```

Or open **Settings → Plugins → Browse Registry** in the smuler app.

## Publish

See [CONTRIBUTING.md](CONTRIBUTING.md).

## Distribution

The registry index is published as a GitHub Release asset:

```
https://github.com/lum1n/smuler-registry/releases/latest/download/registry.json
```

## Security

All entries require Ed25519 signatures, SHA256 integrity checks, and maintainer approval. See [REVIEW.md](REVIEW.md).
